# 任务{#task}

在营销活动工作流中， **[!UICONTROL Task]** 活动允许您指定两种方案：第一个是任务完成时的选择，第二个是任务未完成时的选择（如果手动标记为未完成或已过期）。

![](assets/mrm_task_in_workflow.png)



![](assets/wkf_task_activity.png)
