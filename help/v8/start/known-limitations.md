# 已知限制

* Campaign v8尚不适用于内部部署/混合部署 — 仅作为Adobe管理Cloud Service发布
* 现有客户无法从现有Adobe Campaign环境迁移到Adobe Campaign v8
* 没有双向数据复制，仅从Campaign本地数据库到云数据库
