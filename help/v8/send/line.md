# 创建LINE投放

LINE是免费即时消息、语音和视频呼叫的应用程序，可在所有移动设备和PC上使用。 Adobe Campaign允许您发送LINE消息。


![](../assets/do-not-localize/book.png)请参阅 [Campaign Classic v7 文档](https://experienceleague.adobe.com/docs/campaign-classic/using/sending-messages/line-channel.html?lang=zh-Hans){target=&quot;_blank&quot;}以了解详情。

![](../assets/do-not-localize/book.png) 请参阅 [端到端示例](https://experienceleague.adobe.com/docs/campaign-classic/using/sending-messages/line-channel.html#example--create-and-send-a-personalized-line-message){target=&quot;_blank&quot;}(位于Campaign Classicv7文档中)，以创建并发送您的第一个LINE投放。
