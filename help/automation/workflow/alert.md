# 警报{#alert}



An **警报** 活动向一组操作员发送消息。 它的操作方式与批准活动相同，但在此情况下不需要响应。

![](assets/edit_alerte.png)

警报不是永久性的，因此从控制台中不可见。 分配组的操作员必须具有完整的电子邮件地址才能接收通知。 此活动的配置类似于 **批准**. 用于提醒操作员的默认投放模板为“alertAssignee”。
